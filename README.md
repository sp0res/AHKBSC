# KB-Mouse-AHK

## < ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ >

-- AHK script to allow the use of left shift as middle mouse click and the use ofright mouse click to scroll by dragging a click. --

## Using the Script : 

	Simply execute the script using AHK. 

## AHK Download Page : 

	https://www.autohotkey.com
